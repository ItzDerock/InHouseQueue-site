In-House Queue Commands

Here are all the available commands for InHouseQueue! If you're new here, check out the Set-Up search button!

All Commands

reset - Reset your servers' leaderboard. Remove one person from the queue or ready up. Remove everyone from an ongoing queue.

Usage:

  • /admin reset leaderboard
  • /admin reset user [member]
  • /admin reset queue [gameid]

Examples:

  • /admin reset leaderboard
  • /admin reset user @John
  • /admin reset queue 03134ff5

void - Purge all records of a game. Used if players get "stuck" in a match and cannot queue.

Usage:

  • /admin void [gameid]

Examples:

  • /admin void 03134ff5

change_winner - Change the results of a finished game.

Usage:

  • /admin change_winner [gameid] [team]

Examples:

  • /admin change_winner 03134ff5 Red

winner - Set a winner for an ongoing game without requiring a vote.

Usage:

  • /admin winner [role]

Examples:

  • /admin winner @Red: 316d8cc7

queue_preference - Decide if players can be in multiple queues at once.

Usage:

  • /admin queue_preference [options]

Examples:

  • /admin queue_preference Single Queue

sbmm - Enable/Disable Skill-Based Matchmaking. If Disabled, players can decide on teams with "Switch teams".
When enabled, teams are determined by the advanced MMR algorithm.

Usage:

  • /admin sbmm [options]

Examples:

  • /admin sbmm Disable

setchannel - Set queue channel. You may set as many queue channels as you require.

Usage:

  • /setchannel [#channelname]

Examples:

  • /setchannel #InHouseQueue

setwinnerlog - Game results are sent to this channel. You may only have 1 match history channel per server.

Usage:

  • /setwinnerlog [#channelname]

Examples:

  • /setchannel #match-history

admin top_ten - Set up a Dynamic top-ten leaderboard. Your server must have at least one match played.

Usage:

  • /admin top_ten [#channelname]

Examples:

  • /admin top_ten #leaderboard

help - Display help menu.

Usage:

  • /help

start - Start an InHouse queue. This command can only be run in a queue channel.

Usage:

  • /start

win - Initiate the winner vote. This command can only be run in a lobby channel.

Usage:

  • /win

leaderboard - Display a specific leaderboard.

Usage:

  • /leaderboard [options]

Examples:

  • /leaderboard mmr
  • /leaderboard mvp
  • /leaderboard

rank - Display your rank in the server.

Usage:

  • /rank [options]

Examples:

  • /rank mmr
  • /rank mvp

opgg - Multi op.gg link of Red or Blue team. Players Discord Name must be their IGN or prefixed with IGN:

Usage:

  • /opgg [gameid] [team] [region]

Examples:

  • /opgg 316d8cc7 Red euw
